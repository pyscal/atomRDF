Creating Grain Boundaries#

In this notebook, creation and querying of GBs are discussed. These features are still under active development.

from pyscal_rdf import StructureGraph

g = StructureGraph()

We start by creating a \(\Sigma 5 (3 \bar{1} 0)\)

struct_gb_1 = g.create.defect.grain_boundary(axis=[0,0,1], 
                        sigma=5, 
                        gb_plane=[3, -1, 0],
                        element='Fe')

We can visualise the structure

struct_gb_1.show.all()

struct_gb_2 = g.create.defect.grain_boundary(axis=[1,1,1], 
                        sigma=19, 
                        gb_plane=[-3, 2, 0],
                        element='Fe')

struct_gb_3 = g.create.defect.grain_boundary(axis=[1,1,1], 
                        sigma=19, 
                        gb_plane=[-1, -1, -1],

What are all the samples with symmetric tilt grain boundaries?

res = g.query_sample(g.ontology.terms.pldo.SymmetricalTiltGrainBoundary)
res
SymmetricalTiltGrainBoundary AtomicScaleSample
0 N4e98f9aa26e24d6997cdf675f38a4f30 Sample_1

We see we have one structure in the database. We can also find what is the sigma value of this structure by modifying our query.

res = g.query_sample([g.ontology.terms.pldo.SymmetricalTiltGrainBoundary, 
                     g.ontology.terms.pldo.hasSigmaValue])
res
SymmetricalTiltGrainBoundary hasSigmaValuevalue AtomicScaleSample
0 N4e98f9aa26e24d6997cdf675f38a4f30 5 Sample_1

We can choose the sample, and save it

sample = res.AtomicScaleSample[0]
sample
